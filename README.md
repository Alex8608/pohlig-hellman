# Алгоритм Полига-Хеллмана

Реализация алгоритма Полига-Хеллмана для решения задачи дискретного логарифмирования (DLP) в мультипликативной группе Z/pZ.

## Задача

Дано: простое число `p`, основание `g`, число `h`.
Найти: `x` такое что `g^x ≡ h (mod p)`.

## Реализованный функционал

1. **Решение DLP где порядок — простое число** — метод Шенкса (BSGS) или перебор
2. **Решение DLP где порядок — степень простого** — метод сведения к подгруппам
3. **Решение DLP где порядок — составное число** — редукция Полига-Хеллмана + КТО

## Структура проекта

```
├── utils.py            # расширенный Евклид, mod_inv, factorize, element_order
├── baby_giant.py       # перебор и алгоритм Шенкса (BSGS)
├── crt.py              # Китайская теорема об остатках
├── pohlig_hellman.py   # основной алгоритм
├── main.py             # точка входа
└── Dockerfile
```

## Запуск

```
python main.py <g> <h> <p>
```

Пример:
```
python main.py 6 7531 8101
```

## Запуск через Docker

```
docker build -t pohlig-hellman .
docker run pohlig-hellman python main.py 6 7531 8101
```

## Зависимости

Только стандартная библиотека Python 3.12. Внешних зависимостей нет.
